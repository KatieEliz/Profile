For loop
initializes a variable (i) is less than 5, loop is executed. After each iteration of the loop the value of i is incrimated by 1 (i++).
While loop
The loop variable (i) is intialized to 0 before the loop, (i < 5 ) checks if (i) is less than 5. The loop continues to execute as long as the condition is true. (i++) increments (i) by 1 after each iteration.
Difference between while and for loops
'for' loop is a general purpose loop that allows you to specify an initanilization statment, condition and post statment.
A 'while' loop is a special case of a 'for' loop that omits the initalization and post statements.
